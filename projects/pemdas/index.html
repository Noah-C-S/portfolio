Recursive Pemdas Calculator

The final text game that I want to show off on this page isn't really a game, but a calculator I made in early September 2017. This powerful calculator is capable of parsing out equations from really messy text, such that dfghjk6ghjk+ghjk5gfds returns 11.0 (6+5 is nested in there) and it's capable of PEMDAS, so 2+3*2 = 8.0 while (2+3)*2 = 10.0. To accomplish this, the calculator uses regular expressions for pattern matching and recursion for PEMDAS. It's a great example of my proficency with recursion.

This project may not work well on mobile devices. Tap to dismiss.
