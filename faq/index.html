FAQ

Click me!

You can click on a question to reveal the answer, or you can scroll through them with your scroll wheel if you're on a desktop. I made this cool little faq thing from scratch, originally using jQuery animations, but now it uses CSS3 animations which are better for simple animations like these. If something doesn't work properly and you're using the latest version of your browser, email me at noahcscs+bugs@gmailcom.

What libraries or APIs did you use to make this website?

I used jQuery, Hamburgers by Johnathan Suh and jQuery mousewheel by Brandon Aaron

jQuery was used to make some things, particuarly manipulating the DOM, easier and more concise.

The hamburgers API was used to make the cool animated hamburgers that I use on mobile. If you're unsure of what I'm talking about, lower your browser window's width to under 1000 pixels and it should replace the navigation in the upper right.

And finally, I used jQuery mousewheel to detect mousewheel scrolling without having the page move.

These were primarily used to save myself time while developing this website
